Text objects test file

Words: hello world testing foo bar baz

Quotes: "this is a string" and 'another string' with `backticks`

Parentheses: (some text) and (nested (parentheses) here)

Brackets: [array items] and [more [nested] brackets]

Braces: {object properties} and {nested {braces} example}

Mixed: function("argument", 'string', {key: value})

Try these commands:
- diw = delete inner word (cursor on word)
- daw = delete around word (includes spaces)
- di" = delete text inside quotes
- da" = delete text including quotes
- di( = delete inside parentheses
- da( = delete including parentheses
- di[ = delete inside brackets
- da[ = delete including brackets
- di{ = delete inside braces
- da{ = delete including braces
- ciw = change inner word
- yiw = yank inner word
